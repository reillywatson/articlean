Using the API

In order to use the API, you first need an API key.

Once you've obtained this, calling the API is very simple:

curl http://www.articlean.net/article?apiKey=xxx&inlineImages=[true|false]&url=someUrl

The parameters are as follows:

  • apiKey (required) — your API key.
  • url (required) — the URL of the article to fetch.
  • inlineImages (not required, defaults to false) — if true, images will be returned embedded in the HTML as base64-encoded data. This can make it easier to store articles for offline reading. Note that this functionality is not available at all subscription levels.

Errors

The Articlean API will return the following HTTP response codes on error:

  • 403 — invalid API key. Either sign up to request a key, or email us if you've forgotten your existing one.
  • 408 — request timed out. This generally happens when the requested URL is unresponsive.
  • 500 — unknown error. This likely means the server is down. Send us an email at info@articlean.net.
